Это работающие крестики-нолики в отдельном окне против бота с интеллектом. Написан на Python(Tkinter)
## Как запустить
1. Скачай файл `Tic-Tac-Toe.py`
2. Запусти в консоли командой: `python Tic-Tac-Toe.py`
