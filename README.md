Это работающие примитивные консольные крестики-нолики против бота с интеллектом. Написан на Python, инструкция для игры при запуске кода.
## Как запустить
1. Скачай файл `Крестики нолики.py`
2. Запусти в консоли командой: `python "Крестики нолики.py"`
